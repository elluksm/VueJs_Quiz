# Apraksts

# Arhitektūra

Projekts sastāv no 2 komponentēm:

- Vue.js Frontend projekta
- Laravel API backenda

# Palaišana

Attiecīgajās takās jāpalaiž sekojošas komandas:

## Frontend (quiz-fe)

```
npm install
npm run serve
```

## Backend (quiz-be)

```
php artisan serve
```

Pēc noklusējuma API darbosies uz 8000 porta.
